# Summary Method for SNT Incidence Objects

Provides summary statistics for each incidence level including number of
valid records, mean, median, range, and missing counts, plus overall
data quality metrics.

## Usage

``` r
# S3 method for class 'snt_incidence'
summary(object, ...)
```

## Arguments

- object:

  An object of class `snt_incidence`.

- ...:

  Additional arguments (ignored).

## Value

Invisibly returns a list of summary statistics.
