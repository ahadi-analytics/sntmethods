# Calculate N0 (Crude Incidence) - Internal

Calculate N0 (Crude Incidence) - Internal

## Usage

``` r
.calc_n0_internal(df, scale_factor)
```

## Arguments

- df:

  Data frame with standardized column names

- scale_factor:

  Denominator for incidence rate

## Value

Data frame with n0_cases and n0_incidence columns added
