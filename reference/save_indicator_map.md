# Save Indicator Map

Saves a map to file.

## Usage

``` r
save_indicator_map(map, filename, width = 10, height = 8, dpi = 300)
```

## Arguments

- map:

  ggplot2 or tmap object.

- filename:

  Output file path.

- width:

  Width in inches. Default: 10.

- height:

  Height in inches. Default: 8.

- dpi:

  Resolution. Default: 300.
