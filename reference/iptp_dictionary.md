# IPTp Indicator Dictionary

Returns a tibble describing all IPTp indicators computed by
[`calc_iptp_dhs`](https://ahadi-analytics.github.io/sntmethods/reference/calc_iptp_dhs.md).

## Usage

``` r
iptp_dictionary()
```

## Value

Tibble with columns: indicator, indicator_code, indicator_title,
numerator_description, denominator_description, denominator_code.
